Bioinformatics Software

A protein's amino acid sequence is a large part of how a protein will fold on itself and piece together. Bioinformatics software stores amino acid sequences and simulates how it does or might fold into a 3D structure. Through software string manipulations, different protein structures can be simulated. Then those structures can be vizualized, including being able to rotate structures and zoom in on specific sites. The application of this kind of software is exemplified in the "Drug Discovery" page of this website.

To access and use bioinformatics software, you need a computer and, usually, internet access. There are some online programs like BLAST can be searched and are free to use. Some computer programs, such as BioPython and AutoDock, can be downloaded and installed on your computer. Those computer programs can manipulate and analyze amino acid sequence strings and vizualize proteins' 3D structures.

    Examples of bioinformatics software:

  • BLAST (Basic Local Alignment Search Tool)
  • MAFFT (Multiple Alignment using Fast Fourier Transform)
  • AutoDock
  • GROMACS
  • BioPython
  • BioPerl
  • PyMol
  • R
